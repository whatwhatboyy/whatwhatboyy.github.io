Safety & Security

Important safety information for using gaming tools

Safety Tips

Follow these guidelines to stay safe while modding

Use Antivirus

Keep your antivirus software updated at all times. Scan every file before running it on your system. Use tools like VirusTotal to verify downloads from multiple engines.

Alt Accounts

Always use alternative accounts for modding and cheating. Never risk your main account. Create dedicated alt accounts specifically for testing modifications.

Trusted Sources

Only download from trusted and verified sources. Avoid random links, suspicious websites, and unverified third-party downloads. Stick to known communities.

Keep Updated

Always use the latest versions of cheats and mods. Outdated tools are more likely to be detected and may contain unpatched security vulnerabilities.

Read Instructions

Follow installation guides carefully and completely. Skipping steps can lead to crashes, bans, or security issues. Read all documentation before proceeding.

Backup Data

Always create backups before installing any mods or cheats. Back up game files, save data, and create system restore points so you can recover if something goes wrong.

Antivirus & False Positives

Understanding why gaming tools trigger security warnings

Why False Positives Occur

Cheat software uses injection techniques similar to malware, such as DLL injection and memory modification. This causes antivirus programs to flag them as threats, even when they are legitimate tools from trusted sources.

How to Whitelist

Add the cheat folder to your antivirus exclusion list before extracting files. In Windows Defender, go to Settings > Virus & Threat Protection > Exclusions and add the folder path. This prevents automatic deletion of legitimate tools.

Verify Downloads

Upload suspicious files to VirusTotal.com to scan with multiple engines. A few detections among many clean results typically indicates a false positive. If the majority flag it, avoid the file entirely.
