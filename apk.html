Installation Guides

Step-by-step instructions for installing APK Time

How to Install APK Time on Firestick

  1. Go to Settings
  2. Navigate to My Fire TV/Device
  3. Go down to Developer Options
  4. Click on Apps from Unknown Sources and Click Allow
  5. Go Back Home
  6. Top left corner - click the Search Icon
  7. Type in Downloader
  8. Search and Click on the Downloader App
  9. Click Install, then Open Downloader
  10. Make sure Javascript is enabled in the Settings on Downloader
  11. On Home Page of Downloader, type: letscrackon.org/downloads
  12. Scroll down until you find APK Time
  13. Click APK Time, then scroll down to the Download Button
  14. Once downloaded, click Install then Done
  15. Press Delete and Delete again (this saves storage)
  16. APK Time will now be installed in your Apps Section

How to Install on Other Android Devices

Follow the same steps as shown for installing APK Time on Firestick, but instead of using the Downloader app, you will use a normal web browser (Chrome, Firefox, etc.).

Note: Downloader is for Firestick Only! For Android phones and tablets, simply download the APK using your browser, then open the file to install.

You may need to enable Install from Unknown Sources in your Android device's security settings.

Installation Safety & Guidelines

Enable Unknown Sources

You must enable "Install from Unknown Sources" to install APK files. This is normal for sideloading apps.

Trusted Sources Only

Only download APKs from trusted sources like Archive.org and official repositories. Avoid suspicious third-party sites.

Backup Your Device

Always backup important data before installing new applications, especially on Firestick devices.

Follow Instructions

Carefully follow the installation steps. Skipping steps may cause installation to fail.
