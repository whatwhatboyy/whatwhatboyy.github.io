Frequently Asked Questions

Step-by-step guide to using Cheat Engine

How to Install Cheat Engine

Step 1: Download Cheat Engine from the official website (link in Download section above)

Step 2: Run the installer and follow the installation wizard

Step 3: During installation, be careful to decline any bundled software offers

Step 4: Launch Cheat Engine after installation completes

Note: You may need to disable your antivirus temporarily as it may flag CE as a false positive

How to Load a Cheat Table

Step 1: Download a cheat table (.ct file) for your game

Step 2: Launch your game first and get to the main menu or gameplay

Step 3: Open Cheat Engine

Step 4: Click the folder icon in the top-left corner

Step 5: Browse to and select your downloaded .ct file

Step 6: The cheat table will load and display all available cheats

How to Attach to a Process

Step 1: Make sure your game is running

Step 2: In Cheat Engine, click the computer icon (top-left, glowing icon)

Step 3: A process list will appear - find your game's .exe file

Step 4: Select your game process and click "Open"

Step 5: The window title should now show your game name

Step 6: Now you can activate cheats by checking the boxes in your loaded cheat table

Tip: If you loaded a table first, you may need to click "Yes" when asked if you want to execute the table's script

How to Activate Cheats

Step 1: After loading a table and attaching to your game process

Step 2: Look for checkboxes next to each cheat option

Step 3: Check the box next to the cheat you want to enable

Step 4: Some cheats may have sub-options - click the arrow to expand

Step 5: Return to your game and the cheat should be active

Note: Some tables require you to activate a "main script" or "enable" option first before other cheats work

Troubleshooting Common Issues

Can't find game process: Make sure the game is running and try searching by the game's window name

Cheats not working: Table may be outdated - check for an updated version matching your game version

Game crashes: Disable all cheats and re-enable one at a time to find the problematic one

Antivirus blocking: Add Cheat Engine to your antivirus exceptions/whitelist

Access denied: Run Cheat Engine as Administrator (right-click > Run as administrator)

Pro Tips

Save your game first: Always save before using cheats in case something goes wrong

Use hotkeys: Many tables have hotkeys - check the table description

Read the table notes: Table creators often include important instructions

Join the community: Fearless Revolution and CE forums have helpful communities

Keep CE updated: Newer games may require the latest version of Cheat Engine

Important Information

Single Player Only

Use Cheat Engine only in single-player games or offline modes. Using cheats in online games can result in bans.

Antivirus Warnings

Some antivirus programs may flag Cheat Engine as malicious. This is a false positive - CE is safe when downloaded from official sources.

Backup Your Saves

Always backup your game saves before using cheat tables. Incorrect values can corrupt save files.

Trusted Sources Only

Only download cheat tables from trusted sources like Fearless Revolution or the official Cheat Engine forums.
