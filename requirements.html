Why Do I Need These?

Most of our tools and cheats are built using C# and C++, which require specific runtime libraries to function. Without these dependencies installed, you may encounter errors like:

  • "The application failed to start because .NET Framework is not installed"
  • "VCRUNTIME140.dll was not found"
  • "MSVCP140.dll is missing from your computer"
  • "d3dx9_43.dll" or "xinput1_3.dll" was not found (DirectX errors)
  • "The program can't start because api-ms-win-crt-runtime"

Installing both packages below will ensure all our tools work flawlessly on your system. For DirectX errors, see the troubleshooting section below.

Antivirus Configuration

How to disable antivirus or add exclusions for game tools

Why Antivirus Blocks Game Tools

Game cheats and tools use code injection techniques that antivirus software flags as potentially malicious, even though they're harmless. This is called a false positive.

Two Options:

  • Option 1: Temporarily disable your antivirus (recommended for quick testing)
  • Option 2: Add folder exclusions (recommended for permanent use)

Windows Defender (Windows 10/11)

Most common antivirus - built into Windows

Method 1: Disable Real-Time Protection (Temporary)

  1. Press Windows + I to open Windows Settings
  2. Click on "Privacy & Security" (Windows 11) or "Update & Security" (Windows 10)
  3. Click on "Windows Security"
  4. Click on "Virus & threat protection"
  5. Under "Virus & threat protection settings", click "Manage settings"
  6. Toggle "Real-time protection" to OFF
  7. Confirm the UAC prompt if it appears
Note: Real-time protection will automatically turn back on after a while. This is temporary and good for quick testing.

Method 2: Add Folder Exclusion (Permanent)

  1. Press Windows + I to open Windows Settings
  2. Click on "Privacy & Security" → "Windows Security"
  3. Click on "Virus & threat protection"
  4. Scroll down and click "Manage settings" under "Virus & threat protection settings"
  5. Scroll down to "Exclusions" and click "Add or remove exclusions"
  6. Click "Add an exclusion" → "Folder"
  7. Browse to your tools folder (e.g., C:\Tools or C:\Cheats)
  8. Select the folder and click "Select Folder"
Recommended: Create a dedicated folder like C:\Tools and add it as an exclusion. Extract all your game tools to this folder.

Restore Quarantined Files

  1. Open Windows Security
  2. Click "Virus & threat protection"
  3. Click "Protection history"
  4. Find your file in the list
  5. Click on it and select "Actions" → "Restore"

Other Antivirus Programs

Avast Antivirus

Disable Protection:

  1. Right-click Avast icon in system tray
  2. Select "Avast shields control"
  3. Choose duration (10 min, 1 hour, etc.)

Add Exclusion:

  1. Open Avast → Settings → General
  2. Go to "Exceptions"
  3. Click "Add Exception" → Browse folder

AVG Antivirus

Disable Protection:

  1. Right-click AVG icon in system tray
  2. Select "Temporarily disable AVG protection"
  3. Choose duration

Add Exclusion:

  1. Open AVG → Settings → General → Exceptions
  2. Click "Add Exception"
  3. Browse and select your folder

Norton Antivirus

Disable Protection:

  1. Open Norton
  2. Click "Settings"
  3. Go to "Antivirus" → Turn off Auto-Protect
  4. Select duration

Add Exclusion:

  1. Settings → Antivirus
  2. Click "Scans and Risks" tab
  3. Under "Exclusions/Low Risks", click "Configure"
  4. Add your folder path

McAfee Antivirus

Disable Protection:

  1. Right-click McAfee icon in system tray
  2. Select "Change Settings"
  3. Click "Real-Time Scanning"
  4. Turn it off and select duration

Add Exclusion:

  1. Open McAfee → PC Security
  2. Click "Real-Time Scanning"
  3. Click "Excluded Files"
  4. Add your folder

Bitdefender

Disable Protection:

  1. Open Bitdefender
  2. Go to "Protection" module
  3. Click on "Antivirus"
  4. Toggle "Advanced Threat Defense" off

Add Exclusion:

  1. Open Bitdefender → Protection → Antivirus
  2. Click "Settings" (gear icon)
  3. Go to "Exceptions" tab
  4. Add folder path

Kaspersky

Disable Protection:

  1. Open Kaspersky
  2. Click Settings (gear icon)
  3. Go to "Protection" section
  4. Toggle off "File Antivirus"

Add Exclusion:

  1. Settings → Additional → Threats and Exclusions
  2. Click "Manage Exclusions"
  3. Click "Add" → Select folder

Best Practices

  • Create a dedicated folder: Make a folder like C:\Tools or C:\GameCheats
  • Add exclusion BEFORE downloading: Set up the exclusion first, then download files to that folder
  • Extract to excluded folder: Always extract downloaded files to your excluded folder
  • Use Windows Defender: If possible, uninstall third-party antivirus and use Windows Defender with exclusions
  • Re-enable protection: Always re-enable your antivirus after testing

Troubleshooting

Common issues and solutions

Installation Failed

Solution:

  • Run the installer as Administrator
  • Disable antivirus temporarily
  • Ensure Windows is up to date
  • Close all running applications
  • Free up disk space (need at least 5GB free)

Still Getting DLL Errors

Solution:

  • Restart your computer after installation
  • Reinstall both .NET and VC Runtime
  • Check if you downloaded the correct architecture (x64/x86)
  • Try running the repair option in the installer
  • Manually register DLLs using "regsvr32" command

Which Version Do I Need?

Solution:

  • Install the latest version of .NET (6.0 or higher)
  • Use the All-in-One VC Runtime package to install everything
  • Most modern systems need x64 (64-bit) versions
  • When in doubt, install both x86 and x64

DirectX Errors (d3dx9_43.dll, xinput1_3.dll)

Solution:

  • Download DirectX End-User Runtime from Microsoft
  • Install DirectX 9.0c (required for older games)
  • Update your graphics drivers to the latest version
  • Run Windows Update to get latest DirectX updates
  • Download: DirectX End-User Runtime

Graphics/Display Issues

Solution:

  • Update your GPU drivers (NVIDIA/AMD/Intel)
  • Verify your GPU meets minimum requirements
  • Enable Windows Hardware Acceleration
  • Disable overlays (Discord, GeForce Experience, etc.)
  • Run game/tool in compatibility mode

Antivirus Blocking Files

Solution:

  • Add exception/exclusion for the tool folder
  • Temporarily disable real-time protection
  • Restore quarantined files from antivirus
  • Use Windows Defender instead of third-party AV
  • Check our Safety Tips page

Memory/Performance Issues

Solution:

  • Close unnecessary background applications
  • Check Task Manager for high CPU/RAM usage
  • Run game/tool as Administrator
  • Disable Windows Game Mode (can cause issues)
  • Verify at least 4GB free RAM available

Windows Compatibility Issues

Solution:

  • Right-click tool → Properties → Compatibility
  • Try "Windows 7" or "Windows 8" compatibility mode
  • Enable "Run as Administrator" option
  • Disable fullscreen optimizations
  • Update Windows to latest version (Windows Update)

Access Denied Errors

Solution:

  • Run as Administrator (right-click → Run as admin)
  • Check file/folder permissions
  • Disable User Account Control (UAC) temporarily
  • Move tool to a non-system folder (not C:\Program Files)
  • Ensure you're logged in as Administrator

Network/Connection Errors

Solution:

  • Check your internet connection is stable
  • Disable VPN/Proxy temporarily
  • Add firewall exception for the tool
  • Try a different DNS (Google DNS: 8.8.8.8)
  • Verify game servers are online

Game Not Detected

Solution:

  • Run the game first, THEN launch the tool
  • Ensure game is running in windowed/borderless mode
  • Verify game is fully updated
  • Check tool supports your game version
  • Try running both game and tool as Administrator

Extract/Archive Errors

Solution:

  • Use WinRAR or 7-Zip to extract files
  • Extract to a simple path (like C:\Tools)
  • Avoid paths with special characters or spaces
  • Re-download if archive is corrupted
  • Check available disk space before extracting
