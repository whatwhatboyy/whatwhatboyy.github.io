Quick Start Guide

1 Download Script Pack

Choose a script pack below (Pack 1, 2, or 3) and click the download button. Each pack contains multiple scripts.

2 Extract Files

Extract the downloaded .zip file. You'll find .lua script files inside.

3 Install Scripts

Press Win + R, type %appdata% and press Enter.
Navigate to: YimMenu/scripts
Copy all .lua files into this folder.

4 Load in Game

Launch GTA V and inject YimMenu
Open YimMenu → Go to Settings > Lua Scripts
Select your installed scripts from the list and enable them!

That's it! Your scripts are now ready to use. Scroll down to choose your pack.

YimMenu Script Installation

  1. Download YimMenu - Ensure you have the latest version from official sources
  2. Locate Scripts Folder - Find YimMenu > scripts folder in your installation directory
  3. Download Scripts - Save .lua files to the scripts folder
  4. Load in YimMenu - Use the Scripts tab in YimMenu to load and configure scripts
  5. Configure Settings - Adjust script settings and features as needed
  6. Test Carefully - Always test scripts on alternative accounts first

Safety Guidelines

Use Alt Accounts

Always test scripts on alternative accounts first to avoid main account bans.

Be Subtle

Don't make script usage obvious to other players. Avoid excessive displays.

Stay Updated

Keep scripts updated and check compatibility with latest game versions.

Trusted Sources

Only download scripts from trusted sources to avoid malicious code.
