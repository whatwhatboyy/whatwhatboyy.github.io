YimMenu Script Installation

  1. Download YimMenu - Ensure you have the latest version from official sources
  2. Locate Scripts Folder - Find YimMenu > scripts folder in your installation directory
  3. Download Scripts - Save .lua files to the scripts folder
  4. Load in YimMenu - Use the Scripts tab in YimMenu to load and configure scripts
  5. Configure Settings - Adjust script settings and features as needed
  6. Test Carefully - Always test scripts on alternative accounts first

Safety Guidelines
