Disable BattlEye for GTA 5

Step-by-step guide for every launcher — Rockstar Games Launcher, Steam, Epic Games, and the Xbox App.

What is BattlEye?

BattlEye is the anti-cheat that ships with GTA V. It blocks most mod menus and injectors from loading. Disabling it lets mods run — every method below does the same thing: it tells the game to start with the -nobattleye launch argument.

How to Disable BattlEye

Use the one-click tool, or follow the manual steps for your launcher

Easiest Method

One-Click Auto Tool

Don't want to do it by hand? This tool finds your GTA V folder automatically (Rockstar, Steam, Epic, or Xbox), drops in the commandline.txt with -nobattleye, and deletes BattlEye for you.

Download Auto Tool (.bat)

Right-click → Run as administrator. Windows SmartScreen may warn about an unknown app — click More info → Run anyway. Prefer to do it manually? Follow the steps below.

Rockstar Games Launcher

  1. Open the Rockstar Games Launcher.
  2. Click Settings (top right), then select Grand Theft Auto V in the left sidebar.
  3. Scroll down to Launch Arguments (also called "Command Line Arguments").
  4. Type -nobattleye into the box.
  5. Close Settings and launch the game from the launcher.

Steam

  1. Open Steam and go to your Library.
  2. Right-click Grand Theft Auto V and choose Properties.
  3. On the General tab, find Launch Options.
  4. Type -nobattleye into the box.
  5. Close the window and launch GTA V from Steam.

Epic Games Launcher

  1. Open the Epic Games Launcher.
  2. Click your profile icon (top right) → Settings.
  3. Scroll to Manage Games and click Grand Theft Auto V to expand it.
  4. Tick Additional Command Line Arguments.
  5. Type -nobattleye into the box that appears.
  6. Launch GTA V from the Epic Games Launcher.

Xbox App (Game Pass / Microsoft Store)

You need the Rockstar Games Launcher. The Game Pass / Microsoft Store copy of GTA V still boots through the Rockstar Games Launcher — make sure it's installed and you're signed in to your Rockstar account before you start.

The Xbox App has no launch-arguments box, so you add the argument with a commandline.txt file inside the game folder instead.

  1. Open the Xbox App → Library → click Grand Theft Auto V.
  2. Click the ⋯ menu → Manage → Files → Browse... to open the install folder.
  3. In that folder, create a new text file named commandline.txt — or just download the ready-made one below.
  4. Open it, type -nobattleye and save.
  5. Launch the game. BattlEye will be skipped.

Don't want to make the file yourself? Download a ready-made commandline.txt and drop it straight into your GTA V folder.

Download commandline.txt Or use the Auto Tool (.bat)

Tip: Make sure Windows isn't hiding file extensions, or you'll accidentally create commandline.txt.txt. In File Explorer: View → Show → File name extensions.

Still Not Working? (Works on Any Launcher)

If BattlEye still loads, delete its files directly:

  1. Open your GTA V install folder (where GTA5.exe lives).
  2. Delete the BattlEye folder.
  3. Delete BattlEye_x64.dll if it is present.
  4. Launch the game as normal.

Note: Verifying/repairing game files will restore BattlEye, and so will most game updates. Just repeat the steps.
