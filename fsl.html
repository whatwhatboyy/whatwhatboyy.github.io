Important Notice

Remember: Your friends can join you in these sessions, but nothing saves to your real GTA 5 account. All progress made in FSL mode is temporary and local only.

You do need a mod menu with FSL (Nenyooo download) to be able to join online sessions.

You can play with anyone while using the menu so long as you are both using a menu. 2 Nenyoo players, 1 Nenyoo and 1 Scooby player, or any other mod menu combination with BattlEye disabled. This applies to both Legacy and Enhanced editions. Only players that have BattlEye enabled will not be able to join and you will not be able to join them.

If you want to play with your friend and they don't have a mod menu, get them to use the free version of Nenyooo menu.

FSL is a LOCAL SAVE tool. All it does is convert your online save profile into a local save, it is not some magic tool that lets you play public sessions. It may have worked like that before, a long time ago, but it was never the main reason for the tool and it will not be able to do it again. It is for converting save to a local save only.

Playing on Banned Accounts

FSL allows you to continue playing even if your account has been banned:

  • Rockstar Banned: You can play in solo sessions only. Since FSL uses local save data, Rockstar's ban doesn't prevent you from loading into the game.
  • BattlEye Global Ban: You can still play with other FSL users, modders, and cheaters in their lobbies. As long as everyone has BattlEye disabled, you can join sessions together regardless of BattlEye ban status.

How To Disable BattlEye

Follow these steps carefully to disable BattlEye and use FSL

  1. Make sure BattlEye is disabled in the Rockstar Games Launcher settings.
  2. Input the command -nobattleeye in commandline.txt located in your game directory (EPIC GAMES/STEAM etc.).
  3. Start the Command Prompt as Administrator and enter sc delete BEService to remove BattleEye service completely.
  4. Let the game load to the main menu. Do not load story mode or online.
  5. If auto-loading into story or online mode is enabled, disable it in the game settings. After disabling auto-loading, restart the game.
  6. Once the game is at the main menu, inject N3NY000.
  7. IMPORTANT: AFTER THE DLL IS INJECTED SELECT TO GO TO STORY MODE FIRST.
  8. Once you are in story mode, open N3NY000, select Network > Session Starter and join a public or invite-only session.

Frequently Asked Questions

Common issues and solutions for FSL users

I keep getting desynced from public sessions every five minutes

We currently do not have a BattlEye bypass, and legitimate hosts will eventually remove you due to a heartbeat failure. There is currently no way to stop this other than using an actual (private) bypass. RIPTIDE or Infamous have this, while N3NYOOO needs a recode before it can support BattlEye.

I removed FSL and all my progress disappeared!

FSL reroutes account save data to disk, so any progress made with FSL will only show up if you have FSL enabled. If you don't want this, you can also use NENY000 without FSL.

I removed FSL and the game doesn't start up anymore

This is a known issue; delete Documents/GTAV Enhanced/Profiles to fix. Make sure you have latest FSL file.
