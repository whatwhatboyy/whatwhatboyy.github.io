Image Carousel Example

This demonstrates how to add multiple images to cheat cards with arrow navigation.

How to Use the Carousel System

1. Add the carousel script to your page

<script src="assets/js/image-carousel.js"></script>

2. Add data-carousel-images attribute to your cheat card

<div class="cheat-card" data-carousel-images='[
    {"src": "image1.jpg", "alt": "First image"},
    {"src": "image2.jpg", "alt": "Second image"},
    {"src": "image3.jpg", "alt": "Third image"}
]'>

3. The carousel will automatically be created!

Features

  • Arrow Navigation: Click left/right arrows to navigate between images
  • Indicator Dots: Click dots at the bottom to jump to specific images
  • Touch Swipe: Swipe left/right on mobile devices
  • Auto-advance: Images advance automatically every 5 seconds
  • Hover Pause: Auto-advance pauses when hovering over the carousel
  • Modal Compatibility: Images still open in modal when clicked

Dynamic Management

You can also add or remove images programmatically:

// Add an image to an existing carousel
window.imageCarousel.addImageToCarousel(cardElement, {
    src: "new-image.jpg",
    alt: "New image"
});

// Remove an image from a carousel
window.imageCarousel.removeImageFromCarousel(cardElement, imageIndex);
