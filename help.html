Download & Installation

My antivirus is detecting the cheat as a virus. Is it safe?

This is a false positive and completely normal. Game cheats use code injection techniques that antivirus software flags as potentially malicious behavior, even though they're harmless.

  • All our files are manually scanned and verified safe
  • We only host files from trusted developers
  • Temporarily disable your antivirus during download and extraction
  • Add the cheat folder to your antivirus exclusions
How do I properly install and run the cheats?

Follow these steps for proper installation:

  1. Disable your antivirus temporarily
  2. Download the cheat file to a new folder
  3. Extract all files (use WinRAR or 7-Zip)
  4. Run the executable as Administrator
  5. Start your game and inject the cheat
  6. Follow any specific instructions in the README file
The cheat isn't working or won't inject. What should I do?

Try these troubleshooting steps:

  • Check game version compatibility - The cheat might need to be updated for the newest game version
  • Make sure you're running the cheat as Administrator
  • Check if your game version is compatible
  • Disable antivirus and Windows Defender
  • Close any overlays (Steam, Discord, NVIDIA)
  • Try running the game in Windowed mode
  • Update your cheat to the latest version
I'm getting DLL errors or missing DirectX files. What do I need?

Most cheats and tools require specific system dependencies to function properly. Visit our Requirements Page to download and install:

  • .NET Framework - Required for C# based cheats and tools
  • Visual C++ Runtime (All versions) - Required for C++ based applications
  • DirectX End-User Runtime - Fixes d3dx9_43.dll and xinput errors
  • Updated Graphics Drivers - NVIDIA, AMD, or Intel drivers

Common errors this fixes:

  • "VCRUNTIME140.dll was not found"
  • "MSVCP140.dll is missing"
  • "d3dx9_43.dll" or "xinput1_3.dll" not found
  • ".NET Framework is not installed"
Pro Tip: Install everything from the Requirements Page to avoid all DLL and runtime errors!

How can I protect my account while cheating?

Follow these protection strategies:

  • Use alt accounts only - Never risk your main account
  • Be subtle - Don't make it obvious you're cheating
  • Use private sessions - Avoid public lobbies when possible
  • Don't grief - Avoid harassing other players
  • Gradual progression - Don't add millions instantly
Is it safer to use cheats in single player vs online?

Single Player is always safer than online play:

  • Single Player: Zero ban risk, no anti-cheat detection
  • Online: Higher risk but manageable with proper precautions
  • Some cheats work only in single player (like Menyoo)
  • Online cheats require more sophisticated protection systems
