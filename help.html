Help & FAQ

Common questions about cheats, mods, and safety

Why do I have to disable my antivirus?

Game cheats use code injection and memory manipulation techniques that antivirus software flags as potentially malicious behavior. This is known as a false positive -- the files are not actually harmful, but their behavior patterns resemble those of malware.

To use cheats safely:

  • Temporarily disable your antivirus or Windows Defender before downloading
  • Add the cheat folder to your antivirus exclusion list
  • Re-enable your antivirus after extraction is complete

All files hosted on this site are manually scanned and verified safe before being published.

Are all the cheats safe and undetected?

We test all cheats before featuring them and only list ones that are currently working and undetected. However, no cheat is 100% safe forever. Game developers constantly update their anti-cheat systems, which means:

  • A cheat that is undetected today may get detected in a future update
  • Using cheats in online games always carries some risk of a ban
  • We strongly recommend using alt accounts rather than your main account
  • Single-player cheats are generally much safer than online ones

Always check the cheat's status on our site before using it, as we update detection statuses regularly.

How do I install a mod menu?

Follow these general steps to install most mod menus:

  1. Disable your antivirus temporarily to prevent false positive detections
  2. Download the cheat from the link provided on our game page
  3. Extract the files using WinRAR or 7-Zip to a dedicated folder
  4. Run the executable as Administrator (right-click > Run as administrator)
  5. Launch your game and wait until you are fully loaded in
  6. Inject the cheat by clicking the inject button in the cheat loader
  7. Use the menu key (usually Insert, F5, or Numpad 0) to open the mod menu in-game

Each cheat page includes specific installation instructions. Check the YouTube tutorial linked on the download page for a visual walkthrough.

What should I do if a cheat isn't working?

If a cheat is not working, try these troubleshooting steps in order:

  • Run as Administrator -- right-click the executable and select "Run as administrator"
  • Disable antivirus completely -- Windows Defender, third-party AV, and real-time protection
  • Check game version -- make sure your game is updated to the version supported by the cheat
  • Close overlays -- disable Steam overlay, Discord overlay, NVIDIA GeForce Experience, and MSI Afterburner
  • Re-download the cheat -- your antivirus may have quarantined or corrupted files
  • Try windowed mode -- some cheats work better when the game is not in fullscreen
  • Check our Discord or Telegram -- the cheat may have been detected or updated

Can I get banned for using cheats?

Yes, there is always a risk of being banned when using cheats in online games. Anti-cheat systems are constantly being improved, and no cheat can guarantee permanent undetected status.

To minimize your risk:

  • Always use alt accounts -- never cheat on your main account
  • Be subtle -- avoid obvious cheating like rage aimbotting or teleporting
  • Use private sessions when possible to avoid player reports
  • Do not grief other players -- this attracts reports and manual reviews
  • Add money or items gradually rather than all at once
  • Keep cheats updated -- outdated versions are more likely to be detected

Single-player and offline modes are always the safest option.
